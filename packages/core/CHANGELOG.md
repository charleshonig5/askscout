# askscout-core

## 0.1.0

### Initial release

- First and only published version. Provides the shared library powering the `askscout` CLI and the askscout.dev web app: git reading, LLM digest summarization, output formatting, and project state management.

### Deprecated

- This package is now deprecated. The `askscout` CLI bundles `askscout-core` directly starting with `askscout@0.2.0`. There is no reason for downstream users to install this package separately. Use `npm install askscout` instead.
- The package will remain on npm for historical reference but receives no further updates.
